# AI 接手手册（AI-GUIDE）

这份文档写给**下一个接手迭代的 AI**（以及使用 AI 的你）。目标：让任何模型在 5 分钟内安全上手，不破坏既有约定。

## 1. 快速接手三步

1. 把 `index.html` **和本仓库全部 `.md` 文档**一起发给 AI（缺文档的接手是危险的）；
2. 让它先读 `index.html` 顶部的段落索引 → `ARCHITECTURE.md` 的「硬约定」→ 本文的「红线」；
3. 再提需求。

### 可直接复制的接手提示词

```
你好，请接手迭代单文件应用「趟趟清」（货运趟次记账，给货车司机用）。
我已上传 index.html（应用本体）和全部 .md 文档。

开工前请务必：
1. 通读 index.html 顶部的文件结构索引（[CSS-1]~[JS-6]）；
2. 读 ARCHITECTURE.md 的「硬约定」和「统计口径」——这些不允许改；
3. 读 AI-GUIDE.md 的红线清单和改动流程；
4. 读 DESIGN.md，视觉与文案要保持「宣纸账房 / 瓷青泥金」的既有气质。

本次需求：______________________

交付要求：
- 最小改动，保持单文件、零依赖、零构建；
- 说明你改了哪些段落、为什么这样改；
- 在 CHANGELOG.md 里补一条记录；
- 告诉我如何验证：#test 自检 + TESTING.md 冒烟清单中受影响的步骤。
```

## 2. 红线清单（违反任何一条 = 返工）

1. **不改统计口径**（编号按发车年 / 利润按到家月 / 维修单列）——用户的历史账目语义依赖它们；确需改，先和用户确认，且必须先在 [JS-6] 写好新口径的断言；
2. **不引外部依赖**：没有 CDN、npm、外链字体图片，一个文件走天下；
3. **不加构建步骤**：交付物永远是可直接双击打开的 `index.html`；
4. **数据结构变更必须走迁移**：结构变更升 `SCHEMA_VER` + 写 `migrations`；只加 settings 字段用默认值合并（见 ARCHITECTURE §5）。绝不静默改字段含义；
5. **所有用户输入渲染前过 `esc()`**；
6. **金额入口处两位小数、拒绝负数**；
7. **科目只软删除**（翻 `active`）；
8. 文案遵守 DESIGN.md §6 的语气（口语、动词开头、面向司机）；颜色只用 CSS 变量；
9. 弹层一律走 `openSheet/closeSheet`，不要自己 classList 开关（会破坏历史栈和返回键）；
10. 提交型按钮记得 `guardOnce()` 防连点。

## 3. 改动流程

```
定位段落（用头部索引 + 注释块搜索）
→ 最小 diff，风格贴合现有代码（单引号、无分号省略、字符串拼接不用模板串也可，但保持一致）
→ 跑 #test（涉及 [JS-1]/[JS-2] 时先补断言）
→ 过 TESTING.md 冒烟清单中受影响的条目
→ CHANGELOG.md 记一条
→ 交付时说明改动点与验证方法
```

## 4. 常见任务菜谱

**给支出条目加一个字段（如"地点"）**
[JS-1] defaultData 注释里补字段说明 → migrate 清洗处兜底 → 快记弹层 [HTML-2] 加输入 → quickSave 写入 → 详情渲染 renderTripDetail 展示 → 改一笔的 openAmtPad 配置传入/onSave 写回 → 自检加断言。

**加一个新弹层**
[HTML-2] 仿照现有 sheet 结构（`.sheet > .shead(‹ + h3) + .sbody`）→ 打开用 `openSheet($('#sheet-xxx'))` → 关闭后如需刷新下层，在 closeSheetVisual 的 `re` 映射表里加一项 → 交互走事件委托加 `data-*` 分支。

**加一个设置项**
defaultData().settings 加默认值（不用升 schema）→ 更多页 [HTML-1] 加行 + `data-go` → 委托里处理 → 自检加一条 `migrate(null).settings.xxx` 断言。

**改主题色 / 加颜色**
只动 [CSS-1] 变量或新增变量（两主题都要给值）→ DESIGN.md 令牌表同步。

**加一种报告**
[JS-5] 仿 genReport：拼纯文本行数组 → `lastReport = {title, text}` → 复用 `#sheet-report`。

## 5. 坑清单（前人踩过的）

- 事件委托的 `q()` 匹配是有顺序的，新分支放错位置可能被上面的分支截胡；
- `closeSheet` 之后下层会按 `re` 映射自动重渲染——别在调用点重复渲染造成闪烁;
- `String.prototype.replace` 的字符串替换在本项目补丁脚本里被 split/join 取代过：替换文本含 `$$` 会被吞（`$('#x')` 安全，`$$('.x')` 不安全），写脚本处理本文件时注意；
- iOS 对 `background-attachment:fixed` 支持差，氛围渐变画在 `body::before(position:fixed)` 上，别挪回 body；
- 弹层开着刷新页面会残留历史格（BACKLOG 已登记），别试图在 unload 里"修"它，得不偿失；
- `tripSeq` 是动态编号，任何"把编号存起来"的想法都会破坏补录语义。
