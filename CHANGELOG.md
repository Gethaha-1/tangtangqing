# 趟趟清 · 更新日志（CHANGELOG）

## v1.1.0 · 2026-07-11 iPhone 全屏手势适配

**修复（针对"添加到主屏幕"独立全屏模式）**
- 顶部避开手机状态栏：给主页面与所有弹层头部加 `env(safe-area-inset-top)` 留白，解决标题/返回按钮与时间电量栏重叠
- 修复独立模式下顶部返回按钮被状态栏压住、点不到的问题

**新增手势（提升接近原生 App 的手感）**
- 左缘右滑返回：任意弹层内从屏幕最左缘（28px 内）向右滑 ≥70px 关闭返回，带跟手位移与回弹；竖直漂移超阈判为滚动不误触
- 弹层顶部下拉关闭：从弹层头部下拉 ≥90px 关闭
- 已排除科目条、金额速选、数字键盘、滑动收车等横向/拖拽区域，避免与既有交互冲突

**不变**
- 数据结构、统计口径、账目数据、部署网址均未改动，直接替换文件即可，旧账全部保留

## v1.0.0 · 2026-07-07 首个正式版

**功能**
- 趟次记账：发车 → 在途快记（科目横滑 + 大数字键盘 + 连加 + 常用金额速选 + 可选备注）→ 滑动收车结算，利润自动计算
- 补录旧账：一屏表单，按纸质本子顺序誊总数
- 趟次列表与详情：按年分组、红章/LED 双主题卡片；每一笔可改可删，日期可改
- 统计：年选择、月度利润柱图、费用构成、趟均利润、净落袋
- 维修保养独立账（不计入趟利润，统计与报告单列）
- 月报/年报纯文本生成 + 复制/分享；每月 1 号后首页自动提醒上月报告
- 科目管理：随时新增/改名/停用支出与收入科目（含图标挑选）
- 备份导出 / 恢复导入（JSON，含跨版本迁移）
- 日夜双模式一键切换（日间新账本 / 夜间夜路仪表盘）
- 三级存储降级：localStorage → Claude 预览 storage → 内存（带警告）

**本版设计决策（来自用户确认，改动前先问用户）**
- 不记路线、不记里程/油耗——保持录入极简
- 「装卸费」单科目多笔记（装车/卸车工人小费各记各的）
- 维修保养不进趟次营收，单独立账
- 月/年统计按到家日期归属；趟号按发车年内顺序
- 邮件暂为"生成报告手动分享"，全自动发送留待二期（见 ROADMAP）
